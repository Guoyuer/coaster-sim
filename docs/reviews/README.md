# Reviews

Live reviews should be rare. Put a new file here only when active
external/human feedback can block current work.

Old reviews are archived in `docs/reviews/archive/`. Do not read them during
normal iteration unless `docs/plans/photoreal-progress.md` or
`docs/plans/photoreal-plan.md` points to a specific archived review.

New review files should be concise:

- target screenshot/commit
- verdict: `PASS`, `FAIL`, `PARTIAL`, or `NEEDS-HUMAN`
- blockers
- allowed next actions
- human decisions required
